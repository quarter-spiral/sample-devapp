Friends Bar

Display custom values as badges on each player in the friends bar

You can use the badges to display a player's Level, XP, Score, etc... And since players can sort their friends by them, it can be an instant high score feature!

Consult the Integration Docs to learn how to store (and retrieve) custom values from your game that can be used here.
