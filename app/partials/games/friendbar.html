Friend Bar

Display custom data as badges for each player in the Friend Bar.

For instance, Friend Bar badges can display each player's score, level, xp, rank or anything else you can imagine. Many developers use the Friend Bar as a social high scores table.

Consult the integration docs to learn how to store and retrieve custom data which the Friend Bar can access.
